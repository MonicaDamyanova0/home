Animation

I started animating some time around grade 5 when I was able to mess around with Photoshop. For a while I had wanted to learn how to animate and I had seen other people animating with Photoshop so I though I might as well try to. It was a fun experience and continued animating when I found the time to. I never really finished animating anything though.

When I got a tablet, I started to use it a lot more for animating and digital art in general much more. I would use the free version of FlipaClip and make short, little animations, nothing great though.

When I got an iPad, I bought Procreate and the full version of FlipaClip. That's when I started to try and do more with animation. I would try to finish more animations and do ones which are more difficult. It's a lot of fun and I really want to do more in the future.

Video Editing

Since animations would take a very long time to do entirely by hand, I thought it would be nice to learn how to use a video editing software to be able to do some things quicker. I decided to learn how to use DaVinci Resolve since it's free and looked pretty good. It's pretty fun and easy to use. It's so easy to make videos look good with it and I kind of wish that I learned it sooner.

I haven't made a lot with it yet, but I plan on doing more and have some more animations that I'm currently working on.
