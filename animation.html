Animation

I started animating some time around grade 5 when I was able to mess around with Photoshop. For a while I had wanted to learn how to animate and I had seen other people animating with Photoshop so I though I might as well try to. It was a fun experience and continued animating when I found the time to. When I got an iPad, I started to use it much more for animation since I could take it anywhere I wanted to.

Video Editing

Since animations would take a very long time to do entirely by hand, I thought it would be nice to learn how to use a video editing software ot be able to do some things quicker. I decided to learn DaVinci Resolve since it's free and looked pretty good.

I haven't made a lot with it yet, but I plan on doing more and have some more animations that I'm working on.

Two of My Animations

The Land The Bush
This was a project I made for art a few weeks ago. I drew everything and did all the frame-by-frame animations in Procreate, then put it all together in DaVinci Resolve.
